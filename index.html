Projects

Most recent work

Underwater Image Enhancement Using Deep Learning

This project involves developing a deep learning architecture for enhancing authentic underwater images. In this project, synthetic underwater image data with various water types was used to train the deep neural network. Ten different models were built to handle different types of underwater images. These models were tested on real underwater images.

GitHub repository

No Reference Underwater Image Quality Assessment

This project involves developing a model to rank ten different underwater image enhancement algorithms. In this project, subjective annotated underwater image datasets containing subjective scores were used. Using these datasets, features were extracted from the images, and an SVMRank model was trained to rank the different underwater image enhancement algorithms.

GitHub repository

Sand Dust Image Enhancement

This project aims to enhance sand dust images by performing color balance through color compensation, white balance correction, converting the image from RGB to HSV, enhancing the luminance component using CLAHE, and normalizing the saturation.

GitHub repository

Ecg Based Arrthymia Detection Using RR interval and Average Energy Features

In this project, we conducted preprocessing on ECG data, extracting key features like RR intervals and Average Energy. We then employed Support Vector Machines (SVMs) to classify the data into normal and abnormal ECG readings, achieving a notable accuracy rate of 95%. This successful classification has significant implications for medical diagnostics and underscores the effectiveness of our approach.

GitHub repository

Filtering of Signal Corrupted by Gaussain Noise Using Weiner Filter

In this project, I employed the Wiener filter to denoise signals and computed optimal weights using the Wiener-Hopf equations. Additionally, I analyzed the convergence of the Steepest Descent Algorithm on the error performance surface.

GitHub repository
